## Operadores Aritméticos

Exercícios voltados para o uso de operadores aritméticos e cálculos básicos
em Python.

Conceitos praticados:
- Operadores matemáticos (`+`, `-`, `*`, `/`, `//`, `%`, `**`)
- Precedência de operadores
- Cálculos simples e compostos
- Uso de variáveis em operações matemáticas
