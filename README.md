# Exercícios em Python 

Este repositório reúne exercícios de **lógica de programação e fundamentos em Python**,
desenvolvidos com base no curso **Curso em Vídeo – Python**, do professor Gustavo Guanabara.

O objetivo deste repositório é registrar minha evolução no aprendizado da linguagem,
praticando conceitos fundamentais de forma contínua e organizada.

---

##  Objetivos do repositório

- Praticar lógica de programação
- Consolidar fundamentos da linguagem Python
- Registrar evolução no aprendizado
- Criar um histórico público de estudo e prática

---

##  Conteúdos praticados até o momento

- Tipos primitivos e saída de dados  
- Operadores aritméticos  
- Utilização de módulos  
- Manipulação de strings  
- Estruturas condicionais (`if`, `elif`, `else`)  
- Condições simples e aninhadas  

---

##  Observações

- As aulas iniciais do curso (instalação e introdução) não foram incluídas, pois não envolvem exercícios práticos.
- A numeração dos exercícios segue a numeração original do curso, para facilitar referência e acompanhamento.

---

##  Em constante evolução

Este repositório está em desenvolvimento contínuo e será atualizado conforme novos
conteúdos forem estudados e praticados.
