## Utilização de Módulos

Exercícios focados na utilização de módulos internos do Python
para reaproveitamento de funcionalidades.

Conceitos praticados:
- Importação de módulos com `import` e `from`
- Uso de funções prontas
- Módulos como `math` e `random`





### Exercício 21 – Reproduzir áudio

Este exercício utiliza a biblioteca `playsound3` para reproduzir um arquivo MP3.

Para executar:
1. Instale a biblioteca:
   pip install playsound3
2. Adicione um arquivo MP3 chamado `ex021.mp3` na mesma pasta do script
3. Execute o arquivo `ex021.py`

Obs: o arquivo de áudio não está incluído no repositório.
